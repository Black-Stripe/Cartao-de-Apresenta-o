

# Trabalho Prático N3 - Frontend Design

Este repositório contém o projeto individual desenvolvido para a nota da N3 da disciplina de Fron-tend Design. O objetivo do projeto é estruturar uma página web integrada com serviços externos utilizando a BrasilAPI,e mostrar meus conhecimentos teóricos e praticos aprendidos em aulas com o ministrador WILLIAN HUBNER,
O trabalho consiste em uma especie de cartao de visita,como se fosse um curriculo mais aprimorado e automatizado,com links para os sites e aplicativos pessoais como; github,instagram e wahtsapp

## Funcionalidades
A aplicação realiza o consumo de rotas públicas de forma dinâmica:
1. *Consulta de CEP: Retorna informações de endereço (Cidade, Estado e Bairro).
2. *Consulta de CNPJ: Mostra dados cadastrais como Razão Social, Nome Fantasia e Situação.
3. *Consulta de BANCOS:O usuario faz a solicitação/pedido e os retorno são sempre JSON, tornando fácil integrar com qualquer aplicação uma base de dados atualizada com informações dos bancos oficiais fornecidas pelo Banco Central..
